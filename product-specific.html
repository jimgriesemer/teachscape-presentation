Sidebar Background

Sidebars should use the grid with "col-no-gutter" so that it hugs the left side of the window. To make the sidebar "sidebar-wrap" is used and the menu uses ul and li tags. adding "active" to the li tag will make it dark but a span with "left-arrow" must be inserted after the text.

Using the "tree-#" class will add indentation. The # should be the level in the hierarchy starting with 0. Adding class "active" will show it as selected and with a black bar and the arrow. A HREF tags may or may not be used

  • 2 Classroom Environment
  • 2a Creating an Environment of Respect and Rapport
  • 2b Establishing a Culture for Learning this one is really really long for three lines
  • 2c Managing Classroom Procedures
  • 2d Managing Student Behavior

Title Bars

The following are the call to action bars for tertiary pages. These usually have headline text, sometimes editable inputs. They usually have a call to action on the right. They need to span 100% width with no space on left and right window edges. You can use a nested grid, as included here and vary the number and/or size of the columns to achieve the designed layout for the elements required.

  • Download Display Version
  • Add/Edit Rubric Description
  • Delete Rubric

CF-3.0: Panels & Sections

To group items, panels are used. The panels sometimes have sections that are defined by horizontal lines, other times they have just one object inside. The following are examples of both. Please right click to "inspect element" and copy the markup between the descriptive comments bookended by "begin..." and "end...".

Single Panel

Headline

This is a description text block using the paragraph element. This could be a lot of text, it could be a little bit. If I could write a poem, I would but unfortunately, I don't possess that type of talent.

<!-- begin single panel -->
<div class="panel">
    <div class="panel-content content-single">
        <h3>Headline</h3>
        <p>This is a description text block using the paragraph element.  This could be a lot of text, it could be a little bit.  If I could write a poem, I would but unfortunately, I don't possess that type of talent.</p>
    </div>
</div>
<!-- end panel -->
