CF-1.0: Typography

Custom Focus specific headline spacing variations. The first example shows a structure (admin-wrap has bg color for page, row-admin provides spacing on a bootstrap row) that should immediately follow a navbar (includes spacing for the top areas of the page below the nav)

Page Headline (primary pages)- top of the page when there is NOT a header-bar required.

Welcome [admin-name]

Custom Focus Overview

	
<div class="row row-admin admin-wrap">
	<div class="cf-page-header">
		<h5 class="">Welcome [admin-name]</h5>
		<h2 class="mar-top-20">Custom Focus Overview</h2>
	</div>
</div> <!-- admin-wrap -->

Primary Product Navbar

This contains the links to the pages within the product and for help.

Title Bars

The following are the call to action bars for tertiary pages. These usually have headline text, sometimes editable inputs. They usually have a call to action on the right. They need to span 100% width with no space on left and right window edges. You can use a nested grid, as included here and vary the number and/or size of the columns to achieve the designed layout for the elements required.

Normal Headline

CF-3.0: Panels & Sections

To group items, panels are used. The panels sometimes have sections that are defined by horizontal lines, other times they have just one object inside. The following are examples of both. Please right click to "inspect element" and copy the markup between the descriptive comments bookended by "begin..." and "end...".

Single Panel

Headline

This is a description text block using the paragraph element. This could be a lot of text, it could be a little bit. If I could write a poem, I would but unfortunately, I don't possess that type of talent.

<!-- begin single panel -->
<div class="panel">
    <div class="panel-content content-single">
        <h3>Headline</h3>
        <p>This is a description text block using the paragraph element.  This could be a lot of text, it could be a little bit.  If I could write a poem, I would but unfortunately, I don't possess that type of talent.</p>
    </div>
</div>
<!-- end panel -->

Rubric Sections

The following are used for sections. Add/edit the class "bar-0" in order to add indentation to indicate Sub-Sections. The icons are TBD for a solution, but this solution should only involve css and not additional html elements.

  • Add Same-Level Section
  • Add Lower-Level Section

<div class="panel section-bar bar-0">
    <div class="panel-content content-single">
	<div class="row">
		<div class="col-xs-7">
			<input type="text" class="text-edit text-code" placeholder="1" size:"4"></input>
			<input type="text" class="text-edit text-title" placeholder="Enter Section Title"></input>
			<textarea class="text-edit text-description" placeholder="Enter Description" rows="2"></textarea>
		</div>
		<div class="col-xs-5  pos-relative">
			<button class="btn-score">Score on this section</button><!-- toggle 'active' state to change icon -->
			<button class="btn-round round-edit"><span class="icon-pencil"></span></button>
			<button class="btn-round round-delete"><span class="icon-trash"></span></button>
		</div>
	</div>		
	<!-- Insert dropdown -->
	<div class="row pos-relative">
		<div class="insert-action">
			<div class="btn-group">
			  <button type="button" class="btn btn-xs btn-default dropdown-toggle" data-toggle="dropdown">
			    <span class="before-add"> Insert Row Below</span> <span class="caret"></span>
			  </button>
			  <ul class="dropdown-menu" role="menu">
			    <li><span class="disabled">Add Higher-Level Section</span></li>
			    <li><a href="#">Add Same-Level Section</a></li>
			    <li><a href="#">Add Lower-Level Section</a></li>
			  </ul>
			</div>
		</div>
	</div><!-- end Insert dropdown -->		
    </div>
</div>

CF-4.0: Form Fields

The following are the form fields, dynamic text fields and other input related classess

Dynamic text fields. Responds to hover and focus

<!--begin dynamic text input field -->
<input class="text-edit" placeholder="Enter Title" size="50"></input>

Labels and Badges

For use on all modules of custom focus. The "label-saved" etc classes add color, not shape so they can be used for labels and badges

Saved Published Locked 

  <span class="badge badge-saved">Saved</span>
  <span class="badge badge-published">Published</span>
  <span class="badge badge-locked">Locked</span>
